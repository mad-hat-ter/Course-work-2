# Руководство пользователя

## 1. Назначение программы

**Для куратора:** просмотр личного графика смен, запись на свободные смены в расписании, просмотр статистики и оплаты за выбранный период.

**Для менеджера:** создание расписаний и просмотр статистики всего отдела.

---

## 2. Условия работы программы

### Технические средства

- Компьютер или смартфон с браузером
- Подключение к интернету

### Условия доступа

1. Учётная запись, выданная руководителем
2. Привязка к отделу
3. Статус «Активен» в профиле

### Запуск

1. Открыть приложение в браузере.
2. На странице входа ввести email и пароль.
3. После успешного входа откроется раздел **Профиль** и боковое меню.

Кнопка **Выйти** в меню возвращает на страницу входа.

---

## 3. Работа с приложением

### Главное меню (куратор)

| Пункт меню     | Назначение                                           |
| -------------- | ---------------------------------------------------- |
| Профиль        | Личные данные                                        |
| Мои смены      | Календарь личных смен                                |
| Расписание     | Сетка смен отдела, запись на смену и снятие со смены |
| Моя статистика | Отчёт по сменам и оплате за выбранный период         |

### Профиль

| Действие       | Описание                              |
| -------------- | ------------------------------------- |
| Просмотр       | ФИО, email, телефон, должность, отдел |
| Редактирование | ФИО, email, телефон                   |

### Запись на смену

| Шаг | Действие                                   |
| --- | ------------------------------------------ |
| 1   | Открыть **Расписание**                     |
| 2   | Выбрать расписание                         |
| 3   | Найти свободную ячейку                     |
| 4   | Записать себя на смену кликнув по ячейке   |
| 5   | Удалить себя из ячейки нажатием на крестик |

- Записаться можно только себя.
- Нельзя занять слот, если все места заняты.
- Нельзя дублировать запись на одну и ту же смену.
- До открытия расписания сетка видна, но запись недоступна.
- После закрытия записи можно записаться только на будущие смены, снять себя со смены нельзя.
- Нельзя записаться на смену, время начала которой уже прошло.
- Подсказка о текущем режиме записи отображается над сеткой расписания.

### Мои смены

| Шаг | Действие                                            |
| --- | --------------------------------------------------- |
| 1   | Открыть **Мои смены**                               |
| 2   | Нажать на стрелки ← →, чтобы посмотреть другие даты |

### Моя статистика

| Шаг | Действие                                   |
| --- | ------------------------------------------ |
| 1   | Открыть **Моя статистика**                 |
| 2   | Указать дату начала и окончания периода    |
| 3   | Посмотреть таблицу с статистикой и оплатой |

## 4. Ошибки

| Сообщение                                        | Причина                                 | Действие                                                       |
| ------------------------------------------------ | --------------------------------------- | -------------------------------------------------------------- |
| Неверный email или пароль                        | Ошибка при входе                        | Проверить данные, обратиться к руководителю                    |
| Не удалось связаться с сервером                  | Backend недоступен                      | Проверить интернет, повторить позже, обратиться к руководителю |
| У вас недостаточно прав                          | Нет прав на действие                    | Использовать пункты своего меню                                |
| Запись на смены в этом расписании ещё не открыта | Еще не наступило время открытия записи  | Дождаться открытия записи                                      |
| На смене нет свободных мест                      | Слот заполнен                           | Выбрать другой час или день                                    |
| Куратор уже записан на эту смену                 | Наложение записи                        | Ничего не требуется                                            |
| Нельзя записаться на прошедшую смену             | Время смены уже прошло                  | Выбрать будущую смену                                          |
| Удаление записей на смены сейчас недоступно      | Расписание закрылось для редактирования | Обратиться к менеджеру при необходимости                       |
| Нет смен в выбранном периоде                     | Не было записей на смены                | Выбрать другой диапазон дат                                    |
| Конец периода не может быть раньше начала        | Неверные даты в статистике              | Исправить даты                                                 |
