# Руководство администратора

## 1. Применение

- создание и редактирование расписания смен
- управление профилями сотрудников
- просматр статистики отдела и выгрузка отчёта в CSV
- запись и удаление кураторов на смены вручную

---

## 2. Требования

- Браузер с поддержкой JavaScript
- Подключение к интернету
- Учётная запись с ролью **Руководитель**
- Доступ к веб-приложению

Вход осуществляется по данным выданным руководителем.

---

## 3. Структура данных в системе

### Справочники

| Сущность     | Назначение                                                |
| ------------ | --------------------------------------------------------- |
| Отдел        | Группа сотрудников                                        |
| Должность    | Позиция внутри отдела                                     |
| Тип смены    | Название смены, базовая ставка, порог и повышенная ставка |
| Пользователь | ФИО, email, телефон роль, должность                       |

### Расписание

| Поле                         | Назначение                                                         |
| ---------------------------- | ------------------------------------------------------------------ |
| Период                       | Дата начала и окончания недели                                     |
| Дата и время открытие записи | С какого момента кураторы могут записываться и снимать записи      |
| Дата и время закрытие записи | До какого момента действует полный режим записи                    |
| Смены                        | Ячейки сетки: день, час, тип смены, максимальное число сотрудников |

После даты закрытия куратор может записаться только на будушие смены, убрать запись нельзя.

### Типы смен

| Поле                             | Описание                                                       |
| -------------------------------- | -------------------------------------------------------------- |
| `rate`                           | Базовая оплата за смену                                        |
| `quantity_for_increased_payment` | После этого числа смен в периоде применяется повышенная ставка |
| `increased_payment`              | Повышенная оплата за смену                                     |

## 4. Инструкция

### Добавление куратора

1. Открыть **Управление профилями**.
2. Нажать **Добавить сотрудника**.
3. Заполнить: имя, фамилия, email, телефон, пароль, должность, роль.
4. Сохранить и выдать сотруднику email и паролем.

### Редактирование сотрудника

1. Открыть **Управление профилями**.
2. Найти сотрудника по поиску или фильтрам.
3. Нажать на строку сотрудника, поменять данные, роль или статус «Активен».

### Создание расписания на неделю

1. Открыть **Расписание**
2. Нажать **Добавить расписание**.
3. Указать период (даты начала и конца), время открытия и закрытия записи.
4. Для каждого дня задать часы работы, тип смены и максимальнео число сотрудников на смене.
5. Сохранить и расписание появится в списке у кураторов и менеджеров отдела.

### Редактирование и удаление расписания

1. Открыть **Расписание**
2. Выбрать период в списке.
3. Нажать кнопку редактирования и изменить даты, сетку или период записи.
4. Удаление доступно, если расписание больше не нужно.

### Запись куратора вручную на смену и удаление его со смены

1. Открыть **Расписание**
2. Выбрать период в списке.
3. В ячейке сетки выбрать куратора из списка или нажать на крестик около имени куратора.

### Статистика отдела

1. Открыть **Статистика**.
2. Указать период (дата начала и окончания).
3. При необходимости найти сотрудников по фильтрам: ФИО, отдел, роль.
4. Нажать **Скачать CSV**, чтобы получить выгрузку статистики файлом.

### Роли и доступ

| Роль         | Доступ                                                  |
| ------------ | ------------------------------------------------------- |
| Куратор      | Свои смены, запись в расписании, личная статистика      |
| Менеджер     | Расписание (создание/редактирование), статистика отдела |
| Руководитель | Всё как у менеджера + управление профилями              |

## 5. Ошибки

| Что произошло                    | Причина                                       | Устранение                                                                        |
| -------------------------------- | --------------------------------------------- | --------------------------------------------------------------------------------- |
| Неверный email или пароль        | Ошибка в учётных данных                       | Проверить email и пароль, при необходимости поменять пароль в карточке сотрудника |
| Ошибка, что недостаточно прав    | Роль не позволяет делать днное действие       | Войти под учётной записью Руководителя или Менеджера                              |
| Куратор уже записан на эту смену | Повторная запись                              | Выбрать другую ячейку или снять существующую запись                               |
| На смене нет свободных мест      | Уже записалось максимальное число сотрудников | Увеличить максимальное число при редактировании расписания или выбрать другой час |
| Расписание не найдено            | Удалено или неверный id                       | Обновить страницу, выбрать актуальное расписание                                  |
| Не удалось связаться с сервером  | Backend недоступен                            | Проверить работу сервера, обратиться к программисту                               |
| Пустая статистика                | Нет смен в выбранном периоде                  | Поменять диапазон дат или проверить записи кураторов                              |
| Сотрудник не видит расписание    | Сотрудник другого отдела или неактивен        | Проверить отдел и флаг «Активен» в профиле                                        |
